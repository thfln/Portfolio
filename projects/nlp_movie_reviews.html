🎬 Sentiment Analysis on Movie Reviews

This project compares traditional machine learning models and transformer- based NLP models for classifying movie review sentiment using the IMDB dataset.

📈 Data Sources

  • IMDB Large Movie Review Dataset (binary sentiment classification)

⚙️ Models

  • TF-IDF + Logistic Regression (Baseline)
  • TF-IDF + Random Forest
  • DistilBERT (Fine-tuned)

📊 Key Visuals

.

🔍 Key Findings

  • DistilBERT outperformed traditional models in both accuracy and F1 score.
  • SHAP plots confirmed strong interpretability for traditional models; token attribution was used for BERT.
